# 🖨 Printer Usage Optimization System

A data-driven printer analytics and optimization system developed during my AIML internship at Mphasis Limited.

This project leverages Machine Learning, NLP, SQL, Power BI, Streamlit, and LLM-based recommendations to optimize printer usage, predict failures, and generate cost-saving insights in an enterprise environment.

---

## 🚀 Problem Statement

Large organizations often face:

- Inefficient printer allocation  
- High operational & maintenance costs  
- Unpredictable printer failures  
- Lack of data-driven decision support  

This system provides predictive analytics and intelligent recommendations to address these challenges.

---

## 🧠 Key Features

- 📊 Printer usage analytics dashboard  
- 🔍 Printer failure prediction  
- 💰 Cost savings prediction (SVR model)  
- 🧾 Issue type classification (NLP + KNN)  
- 🤖 LLM-based maintenance recommendations (RAG approach)  
- 🌐 Interactive Streamlit web application
